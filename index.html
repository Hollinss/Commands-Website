Purpose of this Page

Welcome! This page contains a list of linux, gitbash, and vi commands. Please refer to the below sections for the commands and a brief description of their use.
Thanks for visiting!

Linux Commands

cd : Change Directory
mkdir : Make Directory
ls : List Files
pwd : Print Working Directory

Vi Commands

i : Insert Mode
escape key : get out of Insert Mode
shift key + colon key : command prompt for Vi
w : write (in command prompt)
q : quit (in command prompt)
wq : write and quit (in command prompt)

Git Commands

git init : Create repository
git add : add files before commit
git checkout -b : create new branch and checkout
git checkout : switch to branch
git merge : merge this branch to current branch
